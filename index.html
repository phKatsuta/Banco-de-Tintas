Banco de Tintas Orgânicas

O banco de tintas orgânicas desenvolvido pelo curso de GAM da FATEC Jundiaí visa promover soluções sustentáveis para a indústria de tintas, utilizando matérias-primas naturais e eco-friendly. Este projeto inovador não apenas contribui para a preservação ambiental, mas também busca aprimorar a qualidade estética das tintas, mantendo a segurança e a eficiência dos processos de aplicação.

Um Projeto FATEC feito por alunos; para alunos

O projeto do banco de tintas orgânicas foi desenvolvido pelos alunos do curso de ADS da FATEC Jundiaí, que criaram toda a estrutura do sistema. Desde o banco de dados, passando pelo desenvolvimento do front-end até a elaboração dos scripts, os alunos foram responsáveis por todas as etapas, garantindo uma solução completa e funcional para o gerenciamento e catalogação das tintas orgânicas.
